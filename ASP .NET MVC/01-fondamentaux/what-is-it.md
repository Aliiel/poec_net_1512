### Définition 

Framework = ensemble d'outils créés par microsoft pour permettre de développer facilement de manière organisée et pro 

Exemple : 
si on veut construire une maison : 
- les briques => C#
- plan d'architecture => MVC
- outils => framework
- Equipe => développeurs 
=> maison = application web

ASP MVC Core : architecture sur laquelle se baser + un ensemble d'outil 

ASP .NET Core 
- ASP : Active Server Pages (pages server actives)
- .NET : plateforme Microsoft, ensemble de ses techno .NET 

- MVC : 
M => Model : structure des données 
V => View : visu accessible par l'utilisateur 
C => Controller : coordonne les interractions utilisateurs avec le server 

=> façon d'organiser le code 


### Comment ça fonctionne ?

ASP .NET CORE MVC transforme une requête en une page web que l'utilisateur veut voir 

Concretement sur une adresse url : 
la navigateur envoie une requête http au server 
le server = la machine qui héberge notre appli 
il reçoit la requête, analyse l'url
