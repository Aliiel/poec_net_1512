### Les objectifs 

Présenter du contenu en bloc (fiche produit, résumé, dashboard) avec une mise en forme propre et rapide 


### La structure 

- conteneur dans lequel se trouvera la card avec la classe 'card' 
- image : 'card-img-top'
- body : 'card-body'
- titre, texte : 'card-text', 'card-title'
- actions : boutons 'btn'


### Exemple 

```html
<div class="card">
    <div class="card-body">
        <h4 class="card-title">Produit A</h4>
        <p class="card-text">Résumé du produit</p>
        <a classe="btn btn-sm btn-outline-light" href="#">Détails</a>
    </div>
</div>
```
