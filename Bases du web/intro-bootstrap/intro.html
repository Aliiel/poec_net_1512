Présentation de Bootstrap !

Bibliothèque de classes css qui permet de faire interfaces propres et responsives en utilisant des classes déjà existantes.
Elle permet de gérer :

  • la mise en page en colonnes : 'container', 'col', 'row'
  • des composants ui déjà créés : 'navbar', 'card', 'modal'
  • des utilitaires : petites classes pour rapidement ajuster les éléments

Responsive

Faire une page qui ss'adapte à la taille de l'écran (tel, PC, ..) Bootstrap est mobile first : pense en first au tel, puis on améliore ensuite selon les écrans plus grands 'sm' => small => 576px "petits écrans" 'md' => medium => 768px "ecrans moyens" (généralement tablettes) 'lg' => large => 992px "grands ecrans" (petits ordis) 'xl' => extra large => 1200px "tres grands ecrans" (ordis de base) 'xxl' => extra extra large >= 1400px "ecrans xxl" Correspond a des paliers qui disent "a partir de cette largeur on peut utiliser la mise en page plus large" Container : - 'container' : une boite centrée - 'container-fluid' : pour prendre toute la largeur grid : principe de base : 'container' -> 'row' -> 'col' row : "ligne" de la grid col : "colonne" case dans la ligne row bootstrap est découpée en 12 parts (12 colonnes virtuelles) - col-12 : largeur complete de la row qui prend 12 colonnes - col-6 : 6 colonnes sur 12 - col-4 : 4 colonnes sur 12
